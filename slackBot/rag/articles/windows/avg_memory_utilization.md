# Reducing Memory Usage on Your Windows PC

## Resolution Steps:
1. **Open Task Manager:** Press `Ctrl + Shift + Esc`, then click the **Processes** tab and sort by **Memory** (click the column header).
2. **Close what you're not using.** Every open app and browser tab consumes memory, even when idle in the background.
3. **Quit memory-hungry apps first.** Browsers with many tabs, Teams, Outlook, Figma, Zoom, and virtual machines hold onto significant RAM — right-click the process and select **End task**.
4. **Restart your PC if it still feels slow.** Some apps leak memory over time; a restart resets the baseline.

## What's happening
Your PC is juggling more in memory than it comfortably can. When apps compete for limited RAM, Windows starts offloading data to disk to keep up — which is why things feel sluggish, unresponsive, or occasionally freeze. The CPU isn't the bottleneck here; memory pressure is.
